Mapping Exercises

Part One

  1. Open opendata.cityofnewyork.us.
  2. Search for 311 and select the top link, 311 Service Requests from 2010 to Present.
  3. Click View Data to preview the data. Don't download the entire file right now since it is very large.
  4. Click the Filter tab.
  5. Change the condition to Created Date, is after and use the date picker to select 10/1/2021.
  6. Add a new filter condition, set it to Complaint Type, contains. For the value I used Noise but feel free to explore other complaint type values like Rodent.
  7. Go to the Export tab and download a CSV (the top link is the one you want).
  8. Don't worry about using it yet.

Part Two

  1. Login to your account on Social Explorer
  2. Create a new map of the United States.
  3. Click Change Data to change the data shown on the map.
  4. Experiment with different Geography levels for the data.
  5. Add a mask for NYC or just the borough you're interested in.
  6. Use Auto Adjust (in the bottom right below the legend) to adjust the map colors accordingly.
  7. Use the three-line (hamburger) menu under the data to add a new layer. Under Map layers click Upload data and drag and drop your data from the previous part. When prompted to select latitude and longitude columns do so.
  8. Try styling the points.
  9. Add popups to the points.
  10. Get the link to share your map. We may ask to see yours!
  11. If you have time, experiment with the interface a bit. Can you change the map colors? Add annotations? See what else is possible.

Part Three

Open mapping session

  1. Use your own data or some of the following data sources to make another map with Social Explorer:
    • NYC Open Data, I recommend starting with the datasets, filtered to map files, but consider looking at others too
    • NYC Department of City Planning Open Data
    • US Government's open data portal
    • World Bank's data bank
